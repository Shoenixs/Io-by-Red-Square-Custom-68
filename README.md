# IO by Red Square — Custom 68 CAD-файлы

Частичный релиз CAD-файлов для **Custom 68** — оригинальной 68-клавишной механической клавиатуры от Red Square.


---


## О проекте


Этот репозиторий содержит частичный релиз CAD-файлов для **IO by Red Square Custom 68**, разработанной командой Red Square. Файлы предоставляются исключительно в обменном формате (STEP / DXF) для ознакомления сообществом, личной модификации и образовательных целей.


<img width="3000" height="1688" alt="Custom_68_Black_1" src="https://github.com/user-attachments/assets/32105ab1-1f30-4f72-bf82-7179d0fedb3c" />


Включённая геометрия:
- Верхняя и нижняя части корпуса (внешние поверхности)
- 3D модель и чертёж плейта
- Размеры зон гаскетов
- Расположение зоны квик-релиза
- Вырез под daughterboard и USB-разъём

> Внутренняя монтажная геометрия, CAE-расчеты, производственные допуски и детали проприетарной сборки **не включены**.

## Характеристики


### Корпус


| Параметр | Значение |
|----------|----------|
| Материал | Алюминий |
| Анодирование | Белое / Тёмный асфальт |
| Габариты (Ш × Г × В) | 345,56 × 112,2 × 19,5 мм |
| Высота от стола | 19 мм |
| Угол наклона | 5° |
| Вес сборки | ~1,25 кг |
| Сборка корпуса | Quick-release (без винтов) |
| Аккумулятор | 4000 мАч, встроенный |


---


### Mount


| Параметр | Значение |
|----------|----------|
| Тип | Extended Gasket Mount |
| Гаскеты | 18 шт., 3 зоны |
| Комплектация | 22 мягких + 22 твёрдых гаскета, дополнительный набор винтов для Top-mount|


---


### Plate


| Параметр        | Значение     |
|-----------------|--------------|
| Материал плейта | Поликарбонат |
| Толщина плейта  | 1,5 мм       |
| Стандарт        | Cherry MX    |
| Совместимость стабилизаторов   | PCB-mount    |


> Клавиатура полностью следует **стандарту Cherry MX** и совместима с любыми MX-совместимыми компонентами от всех современных производителей, кроме стабилизаторов Typeplus x YIKB (скоба может соприкасаться с поверхностью плейта) и AEBoards Staebies V2.1 (необходимо увеличить отверстия в плейте).


> Обратите внимание, что свитчи производства Kailh серии BOX Midnight не соответствуют Cherry MX-стандарту и могут вываливаться из плейта. При создании плейта под свитчи производства Kailh рекомендуется уменьшать размер отверстия до 13,95 мм. Мы рекомендуем использовать свитчи G3ms и Gateron.


---


###  PCB


| Параметр | Значение |
|----------|----------|
| Толщина | 1.2 мм |
| Поддержка свитчей | 3 & 5 pin |
| Наличие хот-свап сокетов | да |
| MCU | WB32FQ95 |
| Wireless chip | FREQCHIP FR8003A |
| Подключение | USB Type-C (центр) + Беспроводное (2.4gHz + Bluetooth) |
| Bluetooth | 5.1 |
| Daughterboard | Есть, программируемых 2 тогла |
| Подсветка | RGB South-facing |
| Прошивка | QMK / Vial + QMK / VIA (проприетарный) |


> Обратите внимание на пробел: В v1 ревизии PCB проставки под стабилизаторы для плат толщиной 1,6 мм устанавливаются сверху платы из-за расположения SMD-компонентов зарядки аккумулятора и WMCU.
